Brain Tumor MRI Classification

An AI-powered system assisting patients and doctors in quick and accurate diagnosis of brain tumor MRI scans.

Try Model Now Installation

Overview

The model classifies MRI images into different tumor categories and provides comprehensive diagnostic information:

  • Tumor Name Classification
  • Confidence Score (%)
  • Disease Description
  • Medical Impact Analysis

Dataset

Sourced from Kaggle, processed using ImageDataGenerator.
